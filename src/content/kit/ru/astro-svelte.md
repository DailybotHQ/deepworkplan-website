# Пресет Astro / Svelte

Руководство-рассуждение, которое процесс onboard использует для сайтов Astro, опционально с островами Svelte. Это чек-лист, а не шаблон — обнаруженная реальность побеждает.

## Признаки

- `astro.config.*`; опционально `@astrojs/svelte` и компоненты `.svelte`.
- Content Collections в `src/content/` со схемами Zod.

## О чём рассуждать

- Реальные скрипты линтинга, проверки типов (`astro check`), тестирования и сборки — захваченные дословно.
- Схемы коллекций контента и соглашения о гидратации островов (`client:visible` вместо `client:load`).
- Подходящие стеку навыки для компонентов, страниц и записей контента.

## Примечания

Полезно для сайтов на основе контента, использующих островную архитектуру Astro с выборочной интерактивностью.
