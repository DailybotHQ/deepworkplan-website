# dwp-execute

Выполнение существующего Deep Work Plan.

<CmdExecute lang="ru" class="not-prose my-8" />

## Что он делает

`dwp-execute` выполняет существующий план задача за задачей. Он читает весь план, выполняет каждую задачу по порядку, обновляет `PROGRESS.md` и отмечает статус завершения каждой задачи.

## Использование

```
/dwp-execute [plan]
```

## Поведение

1. Прочитать весь план перед началом.
2. Выполнять задачи по порядку.
3. Обновлять `PROGRESS.md` после каждой задачи.
4. Точно отмечать статус задачи.
5. Останавливаться и спрашивать при блокировке.

## Псевдонимы

- `dwp:execute`
