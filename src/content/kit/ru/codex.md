# Адаптер OpenAI Codex

OpenAI Codex поддерживает DWP через командные процедуры в Markdown.

## Уровень поддержки

**Частичная** — основные команды работают; расширенные возможности (команды агентов) недоступны.

## Установка

Команды DWP живут как процедуры в Markdown, которые агент читает при вызове.

## Вызов

Используйте префикс `#`:

```
#dwp-create <goal>
#dwp-execute
```

## Примечания

Возможности командных агентов доступны только для Claude; Codex поддерживает последовательный рабочий процесс.
