# Пресет Generic

Запасное руководство-рассуждение, которое процесс onboard использует, когда не подходит ни один пресет под конкретный стек. Это чек-лист, а не шаблон — обнаруженная реальность побеждает.

## О чём рассуждать

- Определить реальные команды сборки и тестирования из манифеста, Makefile или CI — захваченные дословно.
- Применять общие соглашения; не делать предположений, специфичных для стека.
- Создавать только те навыки, агентов и документацию, которые соответствуют реальным рабочим процессам репозитория.

## Примечания

Запасной вариант, когда не подходит ни Django, ни Vue + Vite, ни Astro/Svelte, ни Node/TS service, ни Python package/CLI.
