# dwp-status

Отчёт о статусе Deep Work Plan без выполнения.

<CmdStatus lang="ru" class="not-prose my-8" />

## Что он делает

`dwp-status` резюмирует прогресс плана, ничего не меняя. Он отчитывается о завершённых, выполняемых и ожидающих задачах.

## Использование

```
/dwp-status [plan]
```

## Поведение

1. Прочитать план и `PROGRESS.md`.
2. Резюмировать завершённые задачи.
3. Резюмировать выполняемые задачи.
4. Резюмировать ожидающие задачи.
5. Отчитаться, ничего не изменяя.

## Псевдонимы

- `dwp:status`
