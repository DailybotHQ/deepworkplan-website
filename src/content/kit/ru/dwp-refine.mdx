# dwp-refine

Доработка существующего Deep Work Plan.

<CmdRefine lang="ru" class="not-prose my-8" />

## Что он делает

`dwp-refine` изменяет существующий план. Он может добавлять, удалять или переупорядочивать задачи, сохраняя завершённую работу и обновляя таблицу задач.

## Использование

```
/dwp-refine [plan]
```

## Поведение

1. Прочитать существующий план.
2. Добавить, удалить или переупорядочить задачи.
3. Сохранить завершённую работу.
4. Обновить таблицу задач.
5. Зафиксировать, что изменилось и почему.

## Псевдонимы

- `dwp:refine`
