# Адаптер Windsurf

Windsurf поддерживает DWP через систему правил и командные процедуры в Markdown.

## Уровень поддержки

**Полная** — Windsurf читает правила проекта и запускает каждую команду dwp-* из соответствующего файла процедуры.

## Установка

Команды DWP живут как процедуры в Markdown, которые агент читает через систему правил Windsurf.

## Вызов

Используйте префикс `#`:

```
#dwp-create <goal>
#dwp-execute
```

## Примечания

Windsurf читает файлы процедур и выполняет полный последовательный цикл Deep Work Plan.
