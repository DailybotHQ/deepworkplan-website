# dwp-refine

Dopracuj istniejący Deep Work Plan.

<CmdRefine lang="pl" class="not-prose my-8" />

## Co robi

`dwp-refine` modyfikuje istniejący plan. Może dodawać, usuwać lub zmieniać kolejność zadań, zachowując ukończoną pracę i aktualizując tabelę zadań.

## Użycie

```
/dwp-refine [plan]
```

## Zachowanie

1. Przeczytanie istniejącego planu.
2. Dodanie, usunięcie lub zmiana kolejności zadań.
3. Zachowanie ukończonej pracy.
4. Aktualizacja tabeli zadań.
5. Odnotowanie, co się zmieniło i dlaczego.

## Aliasy

- `dwp:refine`
