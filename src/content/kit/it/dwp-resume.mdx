# dwp-resume

Riprende un Deep Work Plan interrotto.

<CmdResume lang="it" class="not-prose my-8" />

## Cosa fa

`dwp-resume` continua un piano interrotto. Ricostruisce lo stato da `PROGRESS.md` e dai file delle attività, poi prosegue dalla prima attività incompleta.

## Uso

```
/dwp-resume [plan]
```

## Comportamento

1. Legge `PROGRESS.md` e i file delle attività.
2. Ricostruisce lo stato di esecuzione.
3. Prosegue dalla prima attività incompleta.
4. Aggiorna i progressi man mano che procede.

## Alias

- `dwp:resume`
