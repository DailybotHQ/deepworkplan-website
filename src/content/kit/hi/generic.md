# Generic preset

फ़ॉलबैक तर्क मार्गदर्शिका जिसे onboard प्रवाह तब उपयोग करता है जब कोई स्टैक-विशिष्ट प्रीसेट लागू नहीं होता। यह एक चेकलिस्ट है, टेम्पलेट नहीं — पता लगाई गई वास्तविकता ही जीतती है।

## किस बारे में तर्क करें

- manifest, Makefile, या CI से वास्तविक build और test कमांड का पता लगाएं — हूबहू दर्ज किए गए।
- सामान्य परिपाटियाँ लागू करें; कोई स्टैक-विशिष्ट धारणा न बनाएं।
- केवल वही skills, agents, और docs उत्पन्न करें जो रिपॉज़िटरी के वास्तविक वर्कफ़्लो से मेल खाते हों।

## टिप्पणियाँ

जब Django, Vue + Vite, Astro/Svelte, Node/TS service, या Python package/CLI में से कोई मेल न खाए तो यह फ़ॉलबैक है।
