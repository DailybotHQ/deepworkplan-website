# Пресет Generic

Резервний орієнтир для міркувань, який процес онбордингу використовує, коли жоден пресет для конкретного стека не підходить. Це чек-лист, а не шаблон — виявлена реальність важливіша.

## Про що міркувати

- Виявити справжні команди збірки й тестування з маніфеста, Makefile або CI — зафіксувати дослівно.
- Застосувати загальні домовленості; не робити припущень щодо конкретного стека.
- Згенерувати лише ті скіли, агентів і документацію, що відповідають реальним робочим процесам репозиторію.

## Примітки

Резерв, коли не підходить жоден із Django, Vue + Vite, Astro/Svelte, Node/TS service чи Python package/CLI.
