# dwp-refine

Допрацьовує наявний Deep Work Plan.

<CmdRefine lang="uk" class="not-prose my-8" />

## Що це робить

`dwp-refine` змінює наявний план. Він може додавати, вилучати чи змінювати порядок задач, зберігаючи виконану роботу й оновлюючи таблицю задач.

## Використання

```
/dwp-refine [plan]
```

## Поведінка

1. Прочитати наявний план.
2. Додати, вилучити чи змінити порядок задач.
3. Зберегти виконану роботу.
4. Оновити таблицю задач.
5. Зафіксувати, що змінилося й чому.

## Аліаси

- `dwp:refine`
