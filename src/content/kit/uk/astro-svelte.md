# Пресет Astro / Svelte

Орієнтир для міркувань, який процес онбордингу використовує для сайтів на Astro, за потреби зі Svelte-островами. Це чек-лист, а не шаблон — виявлена реальність важливіша.

## Сигнали

- `astro.config.*`; за потреби `@astrojs/svelte` і компоненти `.svelte`.
- Колекції контенту в `src/content/` зі схемами Zod.

## Про що міркувати

- Реальні скрипти лінтингу, перевірки типів (`astro check`), тестування й збірки — фіксуються дослівно.
- Схеми колекцій контенту й домовленості щодо гідратації островів (`client:visible` замість `client:load`).
- Доречні для стека скіли для компонентів, сторінок і записів контенту.

## Примітки

Корисний для контентних сайтів, що використовують острівну архітектуру Astro з вибірковою інтерактивністю.
