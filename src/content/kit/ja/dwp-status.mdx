# dwp-status

Deep Work Plan の状態を実行せずに報告します。

<CmdStatus lang="ja" class="not-prose my-8" />

## 何をするか

`dwp-status` は変更を加えずに計画の進捗を要約します。完了済み、進行中、保留中のタスクを報告します。

## 使い方

```
/dwp-status [plan]
```

## 振る舞い

1. 計画と `PROGRESS.md` を読む。
2. 完了済みのタスクを要約する。
3. 進行中のタスクを要約する。
4. 保留中のタスクを要約する。
5. 何も変更せずに報告する。

## エイリアス

- `dwp:status`
