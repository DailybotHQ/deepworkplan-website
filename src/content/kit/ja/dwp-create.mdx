# dwp-create

高レベルの目標から新しい Deep Work Plan を作成します。

<CmdCreate lang="ja" class="not-prose my-8" />

## 何をするか

`dwp-create` は高レベルの目標を受け取り、それを構造化された計画へと変えます。目標を分析し、必要なら明確化のための質問をし、作業を逐次的なタスクへと分解し、計画ファイルを書きます。

## 使い方

```
/dwp-create <goal>
```

## モード

`dwp-create` は既定で**ガイド付き**モードです。コンテキストを収集し、一つの洗練されたドラフトを書き、**それをあなたに見せ、最終的な計画を生成する前に確認を求めます**。これにより、計画が作成される前に必ずあなたが読むことになります。これが推奨される既定であり、特に最初の数回はそうです。

確認を飛ばすには、最後の語として **`trust`**（または `auto`）を付け加えます。エージェントは洗練されたドラフトを書き、そのまま最終的な計画へ進みます。

```
/dwp-create <goal> trust
```

trust モードは、フローを信頼できるようになったうえでのパワーユーザー向けの近道です。ガイド付きモードは既定で人をループの中に保ちます。

## 振る舞い

1. 目標とリポジトリのコンテキストを分析する。
2. 曖昧なら明確化のための質問をする。
3. 洗練されたドラフトを `.dwp/drafts/` に書く（ガイド付きモードでは、レビューして確認する)。
4. 逐次的なタスクへと分解し、README、タスクファイル、PROGRESS を書く。
5. 三つの必須最終タスク（Security Review、Skills & Agents Discovery、Executive Report）で締めくくる。

## エイリアス

- `dwp:create`
