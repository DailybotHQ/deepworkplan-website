# Django プリセット

onboard フローが Django プロジェクトに用いる推論ガイドです。テンプレートではなくチェックリストであり、検出された現実が優先します。

## 手がかり

- `manage.py` と `settings.py`。`models.py`、`views.py`、`migrations/` を持つアプリ。
- パッケージマネージャーとテストランナーは、存在するマニフェストとロックファイルから推測する。

## 推論すべきこと

- 実際のテストコマンド（多くは `python manage.py test` または `pytest`）と、あらゆるリント／型チェックのゲート。原文どおりに捕捉する。
- アプリとマイグレーションの規約、設定と環境のレイアウト、管理コマンドのパターン。
- アプリ、モデル、ビュー、マイグレーションのための、スタックに合ったスキル。

## 補足

標準的なプロジェクトレイアウトに従う Django アプリに有用です。
