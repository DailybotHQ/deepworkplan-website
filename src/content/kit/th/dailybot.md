# แอดออน Dailybot

แสดงความคืบหน้าของ Deep Work Plan ให้ทีม Dailybot เห็น เป็นแอดออนแบบสมัครใจสำหรับทีมที่ใช้ Dailybot อยู่แล้ว

## สิ่งที่เพิ่มเข้ามา

- ตัวเลือกในการติดตั้ง Dailybot agent skill และ/หรือ Dailybot CLI ผ่านขั้นตอนขอความยินยอมของเครื่องมือนั้นเอง
- รายงานความคืบหน้าหรือหมุดหมายแบบ best-effort ที่ร้อยเข้ากับการดำเนินแผน เพื่อให้การทำแผนเสร็จปรากฏต่อทีม

## พฤติกรรม

แอดออนนี้ **ส่งต่อหน้าที่** ไม่สร้างซ้ำของเดิม ตัว Dailybot agent skill เป็นเจ้าของขั้นตอนติดตั้ง การขอความยินยอม การยืนยันตัวตน และสไตล์การเขียน ส่วนแอดออนนี้เพียงเสนอให้ติดตั้งและร้อยขั้นตอนรายงานที่เป็นทางเลือกเข้าไป รายงานนี้เป็นแบบ best-effort อย่างเคร่งครัด และจะไม่ขัดขวางงานหาก Dailybot ไม่มีอยู่ ยังไม่ยืนยันตัวตน หรือเข้าถึงไม่ได้

## หมายเหตุ

ระเบียบวิธี Deep Work Plan แกนกลางไม่มีการพึ่งพา Dailybot **เลย** ควรเสนอเฉพาะเมื่อนักพัฒนาหรือทีมใช้ Dailybot อยู่แล้ว ห้ามติดตั้งให้ทุกคนโดยอัตโนมัติ
