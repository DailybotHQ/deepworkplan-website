# ก่อนและหลัง

การเปรียบเทียบพฤติกรรมของ agent อย่างเป็นรูปธรรม ทั้งเมื่อไม่มีและเมื่อมี DWP

<UnguidedVsDwp lang="en" class="not-prose my-10" />

## ก่อนมี DWP

เมื่อได้รับโจทย์ "เพิ่มการยืนยันตัวตนให้แอป" agent จะ

- เริ่มแก้ไฟล์ทันที
- ลืมไปว่าตนเองแก้อะไรไปบ้าง
- สร้างผลลัพธ์ที่ไม่สม่ำเสมอและตรวจทานได้ยาก
- ทำงานต่อไม่ได้หากถูกขัดจังหวะ

## หลังมี DWP

เมื่อได้รับเป้าหมายเดียวกัน agent ที่ขับเคลื่อนด้วย DWP จะ

- สร้างแผนทั้ง schema, endpoint, middleware, การทดสอบ และเอกสาร
- ดำเนินงานทีละ task พร้อมอัปเดตความคืบหน้า
- สร้างการเปลี่ยนแปลงทีละขั้นที่ตรวจทานได้
- ทำงานต่อได้อย่างหมดจดหลังถูกขัดจังหวะ

## ผลลัพธ์

โมเดลตัวเดียวกันสร้างงานที่น่าเชื่อถือและตรวจทานได้มากขึ้นเมื่อถูกชี้นำด้วยแผน
