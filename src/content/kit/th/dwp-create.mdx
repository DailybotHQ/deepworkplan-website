# dwp-create

สร้าง Deep Work Plan ใหม่จากเป้าหมายระดับสูง

<CmdCreate lang="en" class="not-prose my-8" />

## สิ่งที่มันทำ

`dwp-create` รับเป้าหมายระดับสูงมาแล้วเปลี่ยนเป็นแผนที่มีโครงสร้าง มันวิเคราะห์เป้าหมาย ถามคำถามเพื่อความชัดเจนหากจำเป็น แยกงานออกเป็น task ตามลำดับ และเขียนไฟล์แผน

## วิธีใช้

```
/dwp-create <goal>
```

## โหมด

`dwp-create` ใช้โหมด **guided** เป็นค่าเริ่มต้น มันรวบรวมบริบท เขียนฉบับร่างที่ปรับแต่งแล้วเพียงฉบับเดียว **แสดงให้คุณดู และขอการยืนยันก่อนสร้างแผนฉบับสุดท้าย** คุณจึงได้อ่านแผนเสมอก่อนที่มันจะถูกสร้าง นี่คือค่าเริ่มต้นที่แนะนำ โดยเฉพาะในไม่กี่ครั้งแรก

ต่อท้ายด้วย **`trust`** (หรือ `auto`) เป็นคำสุดท้ายเพื่อข้ามการยืนยัน agent จะเขียนฉบับร่างที่ปรับแต่งแล้วและไปยังแผนฉบับสุดท้ายโดยตรง

```
/dwp-create <goal> trust
```

โหมด trust เป็นทางลัดสำหรับผู้ใช้ระดับสูงเมื่อคุณไว้ใจกระบวนการแล้ว ส่วนโหมด guided คงให้มนุษย์อยู่ในวงรอบการตัดสินใจเป็นค่าเริ่มต้น

## พฤติกรรม

1. วิเคราะห์เป้าหมายและบริบทของ repository
2. ถามคำถามเพื่อความชัดเจนหากคลุมเครือ
3. เขียนฉบับร่างที่ปรับแต่งแล้วไปยัง `.dwp/drafts/` (ในโหมด guided ให้ตรวจทานและยืนยัน)
4. แยกออกเป็น task ตามลำดับ แล้วเขียน README ไฟล์ task และ PROGRESS
5. ปิดท้ายด้วย final task บังคับสองตัว

## ชื่อแทน
