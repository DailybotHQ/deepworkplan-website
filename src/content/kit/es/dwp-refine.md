# dwp-refine

Refina un Deep Work Plan existente.

## Qué hace

`dwp-refine` modifica un plan existente. Puede agregar, quitar o reordenar tareas mientras preserva el trabajo completado y actualiza la tabla de tareas.

## Uso

```
/dwp-refine [plan]
```

## Comportamiento

1. Lee el plan existente.
2. Agrega, quita o reordena tareas.
3. Preserva el trabajo completado.
4. Actualiza la tabla de tareas.
5. Registra qué cambió y por qué.

## Alias

- `dwp:refine`
