# Preset de Vue

Valores por defecto conscientes de la pila para proyectos Vue.

## Qué adapta

- Comando de prueba: `npm test`
- Convenciones de componentes de un solo archivo
- Patrones de la Composition API
- Gestión de estado con Pinia

## Notas

Útil para apps Vue 3 que usan la Composition API.
