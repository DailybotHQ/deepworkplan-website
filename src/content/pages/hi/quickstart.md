## Quickstart

स्किल स्थापित करें, अपनी रिपॉज़िटरी को ऑनबोर्ड करें, फिर किसी भी एजेंट के साथ योजना बनाएँ और निष्पादित करें — वे चरण जो आपकी रिपॉज़िटरी को स्पेक-ड्रिवन और एजेंट-संचालन योग्य बनाते हैं।

## अपनाने का मार्ग

### 1. Deep Work Plan स्किल स्थापित करें

स्किल को अपनी रिपॉज़िटरी में जोड़ें। यह एक राउटर और आठ सब-स्किल्स भेजती है — create, execute, refine, resume, status, verify, onboard, और author। सबसे तेज़ रास्ते के लिए Skills CLI का उपयोग करें:

```bash
npx skills add DailybotHQ/deepworkplan-skill
```

या रिपॉज़िटरी क्लोन करें और जहाँ git तथा एक shell उपलब्ध हों वहाँ setup चलाएँ:

```bash
git clone https://github.com/DailybotHQ/deepworkplan-skill.git && cd deepworkplan-skill && ./setup.sh
```

### 2. रिपॉज़िटरी को ऑनबोर्ड करें

onboard सब-स्किल चलाएँ और एजेंट को अपनी वास्तविक रिपॉज़िटरी पर तर्क करने दें — उसका स्टैक, पैकेज मैनेजर, और वास्तविक सत्यापन कमांड्स:

```bash
/deepworkplan-onboard
```

यह `AGENTS.md`, एक `docs/` ज्ञान-आधार, प्रति-मॉड्यूल docs, और एक क्रॉस-एजेंट `.agents/` घर (`.claude → .agents` सिमलिंक के साथ) उत्पन्न करता है, पतले `dwp-*` कमांड्स जोड़ता है, और योजनाओं तथा ड्राफ़्ट के लिए एक gitignored `.dwp/` तैयार करता है। कुछ भी टेम्पलेट से नहीं लिया जाता; सब कुछ आपकी रिपॉज़िटरी के अनुरूप ढाला जाता है।

### 3. किट विकसित करें और ऐडऑन स्वीकार करें

स्टैक-उपयुक्त स्किल्स, एजेंट्स और कमांड्स विकसित करने के लिए `/skill-create` और `/agent-create` (author सब-स्किल) का उपयोग करें। ऑनबोर्डिंग पाँच ऑप्ट-इन ऐडऑन भी प्रदान करती है — devcontainer, Dailybot, dependency-upgrade, design-system और AI Diff Reviewer — जिन्हें आप तभी स्वीकार करते हैं जब वे उपयुक्त हों। एक रिपॉज़िटरी शून्य ऐडऑन के साथ पूर्णतः अनुरूप होती है।

### 4. योजना बनाएँ और निष्पादित करें

एक Deep Work Plan उत्पन्न करें और इसे कार्य-दर-कार्य चलाएँ:

```bash
/dwp-create <goal>
/dwp-execute
```

काम आगे बढ़ने पर `/dwp-status`, `/dwp-refine`, और `/dwp-resume` का उपयोग करें। हर योजना क्रमांकित कार्य, सत्यापन-गेट, और एक पूर्णता प्रोटोकॉल साथ रखती है ताकि काम सत्रों के पार संरचित और फिर से शुरू होने योग्य बना रहे।

## परिणाम

आपकी रिपॉज़िटरी स्पेक-ड्रिवन और एजेंट-संचालन योग्य बन जाती है: योजना सत्य का टिकाऊ स्रोत है, और रिपॉज़िटरी स्वयं वह हार्नेस बन जाती है जिसके विरुद्ध कोई भी एजेंट चलता है।
