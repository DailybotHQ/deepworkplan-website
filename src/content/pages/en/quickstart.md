## Quickstart

There are three ways to bring Deep Work Plan into your workflow:

- **Install the skill** (recommended) — `npx skills add DailybotHQ/deepworkplan-skill`. One command installs the cross-agent skill pack for Claude Code, Cursor, Codex, Gemini, and Copilot.
- **OpenClaw** — `openclaw skills install deepworkplan`. The pack loads natively on every eligible session.
- **Git clone and setup** — clone the repository and run `./setup.sh`. Any agent that reads Markdown can run a plan.

After installing, point your agent at any repository and let it make the repo AI-first.
