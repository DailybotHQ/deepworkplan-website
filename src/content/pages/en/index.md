## Cal Newport's Deep Work, applied to AI agents

Deep Work Plan (DWP) is an open, MIT-licensed methodology and reference kit for running structured, validated, resumable multi-hour work with AI coding agents. It turns a goal into an agreed plan, breaks that plan into atomic and independently verifiable tasks, and runs each task in a focused loop that ends with an explicit check.

DWP is framework- and agent-agnostic. The plan, the tasks, and the running log are all plain Markdown — readable, reviewable, and version-controlled. No runtime, no lock-in.
